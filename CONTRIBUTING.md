# 参与贡献

欢迎 PR！无论是修复 bug、添加文档还是扩展新技能。

---

## 当前优先级

### 高优先级

- **添加 JavaScript/TypeScript 屎山技能**
  - Python 版已完成，JS/TS 版期待中
  - 可以参考 `skills/shit-code-generator/SKILL.md`
  - 风格要求：超长函数、内嵌轮子、lambda 滥用、环境混乱

- **改进现有 skill 文档**
  - 规则更清晰、例子更典型

### 中优先级

- **添加更多语言的屎山示例**
  - Java，Kotlin，C++都适合做屎山

---

## 提交 PR 流程

1. Fork 本项目
2. 创建你的分支 (`git checkout -b feature/xxx`)
3. 提交你的更改
4. 推送到你的 Fork
5. 打开 Pull Request

---

## 代码规范

- Skill 文件放在对应目录下
- 示例代码放在 `demo/` 目录下
- 更新 README 如果添加了新内容
- 保持项目幽默隐晦的风格

---

开 issue 讨论，或者直接在 PR 里问。
